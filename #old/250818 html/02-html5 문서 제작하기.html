html5 문서의 기본 구조 작성하기와 주의사항
1. 첫 줄에 DTD(문서형 정의선언) 선언 : 문서의 버전을 명시하며, 브라우저가 HTML문서를 분석, 번역하여 표시하고자 할 때 올바른 버전에 맞게 보여주고자 할 때 반드시 명시해야하는 선언문이다. !DOCTYPE HTML

2. 휴면랭귀지 선언 : LANG='KO', 문서 안에 들어간 주로 사용하는 언어를 표시하는데, 웹서버에서 정보 검색 시 문서 안의 내용(한글)이 우선 검색이 되도록 하기 위함이며, AI툴을 통해 소프트웨어가 내용을 정확하게 이해하고 보여줄 수 있도록 하기 위함이다.

3. META : 메타 태그의 역할은 문서의 내용이자 저자, 날씨 정보, 키워드 등을 나타내고자 할 때 사용하며, 인코딩 UTF-8(문자표현코드)를 넣어서 문자의 호환성을 높여준다.

4. 문서 작성 시 주의사항 - HTML 문서는 태그 명령어로 작성 되는데, 영문 대,소문자로 작성할 수 있다.
- HTML 문서는 문자와 문자 사이 띄어쓰기가 기본 1칸이며, 2칸 이상 띄어쓰기 할 때는 $nbsp; 코드를 삽입한다.
- HTML 태그 명령어는 문서 안에서 enter키를 눌러도 줄바꿈이 일어나지 않는다. 줄바꿈을 위해서는 'br' 태그를 사용한다.(line break의 약자)
- HTML 문서 내에서는 특수문자를 입력하고자 할 때 enfity code, charactor code를 검색하여 찾아 입력을 한다. 브라우저가 올바른 해석을 하여 전달하고자 할 때 필요하다. ☆●♥ ⓒ=>♥ ©
- 키보드에 없는 특수문자는 유니코드 변환 사이트를 참고하여 찾아서 삽입한다. (http://kor.pe.kr/convert.htm)
- 주석 : 문서 안에서 설명이나 작성자, 날짜를 메몸한 듯이 적고자 할 때 사용하며 html 주석을 작성한다.
