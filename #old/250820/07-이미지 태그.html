07 이미지 삽입 태그

웹 문서 안에 들어가는 이미지의 종류로는 jpg(고화질 사진), png(고화질, 로고, 아이콘), gif(단순한 색상의 이미지나 움직이는 이미지, 아이콘), bmp, webp, svg의 종류가 있다.

이러한 이미지 종류들을 웹 문서에 삽입하기 위한 태그 요소가 img 태그이며, source의 약자인 'src' 속성과 이미지가 나오지 않을 때 대체할 문자인 'alt' 속성을 사용한다.

작성법) img src='이미지 파일 경로나 주소' alt='이미지 대체 문자'

1. 이미지 삽입하기


URL - Uniform Resource Locator(웹서버에 위치한 파일, 문서의 정확한 경로나 주소, 위치)

URL 주소는 절대주소와 상대주소가 있다.

절대주소는 http://~로 시작하는 주소를 말하며, 상대주소는 현재 문서를 기준으로 주소가 결정되는 주소를 말함.

./폴더명/파일명.확장자 : 문서를 기준으로 폴더가 보일 때 작성함(하위폴더)

../폴더명/파일명.확장자 : 문서를 기준으로 폴더가 다른 폴더일 경우(상위폴더)

파일명.확장자 : 문서와 이미지가 같은 폴더 안에 있을 경우(파일명.확장자만 작성함)

2. alt 속성

alternate text의 약자로서 이미지가 나오지 않을 때 사용자가 이해할 수 있는 대체문자를 말하며, 시각장애인이 스크린 리더를 통해서 음성으로 이해할 수 있는 문자임. 웹접근성을 위하여 반드시 삽입함.

3. 이미지 준비가 되지 않았을 경우 대체할 방법

placehold.co : 간단하고 빠르며 무료인 이미지 플레이스홀더 서비스입니다.
