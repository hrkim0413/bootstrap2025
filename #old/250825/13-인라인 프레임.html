13 인라인 프레임

인라인 프레임(아이프레임)이란? 문서 안에 또 다른 문서를 삽입하기 위한 태그 요소이며 회원가입 시 나오는 페이지의 '약관 내용', '회사 소개 페이지의 실시간 지도', '유튜브의 영상 삽입' 등 원하는 페이지를 삽입하여 하나의 문서처럼 보이게 할 수 있다.

작성법) iframe src='삽입하고자 하는 문서명 또는 주소(경로)'

1. 현재 문서 안에 다른 html 문서 삽입하기

2. 서브웨이 이용약관

3. 구글지도 api 서비스 삽입하기

4. 유튜브 영상

영상을 홈페이지 올릴 때는 직접 가지고 있는 영상 파일을 video 태그를 활용하여 업로드 할 것인지, 유튜브에 업로드하여 iframe으로 삽입할 것인지를 고민하고 사용할 것.
