14 폼(form) 태그

폼 태그란? 사용자가 입력한 정보(아이디, 패스워드)를 서버에 전송하는 역할을 하며, 폼 태그를 활용한 양식의 종류에는 로그인 폼, 회원가입 폼, 검색 폼, 예약 폼, 주문결제 등의 다양한 양식을 만들 수 있다.

폼 태그의 종류
form
input
select, option
textarea
label
button
fieldset
legend

폼 태그 사용하기

legend, 필드 제목 1. 텍스트 박스(6~20자) : 문자를 입력하기 위한 박스로서 '아이디, 검색, 주소, 문의하기 제목 ...'


2. 패스워드 박스(6~20자) : 비밀번호, 비밀번호 확인, 주민번호, 카드번호, 개인정보 작성 시 글자가 보여지지 않게 기호(*)로 표시됨


3. 라디오 버튼 : 여러개의 항복 중에 하나만 선택이 가능한 양식(동의함, 동의안함, 성별 선택, 열차 종류 선택, 좌석 선택, 시간 선택 등)

라디오 버튼에는 각각 id값을 부여하여 이름을 다르게 설정하여야 여러 항목 중에 하나만 선택이 가능, label for 속성의 역할은 라디오 버튼이 아닌 '문자'를 클릭해도 메뉴가 선택이 되도록 하기 위함이다.


4. 체크박스 : 여러개의 항목들 중에서 사용자가 원하는 항목만 선택할 수 있는 약식이며, 다중 선택이 가능.
나의 취미 :

5. 선택 목록 : 쇼핑몰 옵션 선택, 전화번호 입력, 통신사 선택, 예매 목록, 인원수, 언어 설정, 패밀리 사이트 등의 양식을 만들고자 할 때 사용하는 양식


6. 다중 텍스트 박스 : 한줄 입력이 아닌 여러줄의 텍스트 내용을 입력하고자 할 때 사용. (게시판 게시글 작성, 이메일 내용 작성, 댓글 작성, 이용약관)


7. 파일첨부 : 게시판이나 이메일 양식에서 상대방에게 파일을 첨부하여 보내기 위한 양식

버튼 클릭 시 자동으로 파일을 첨부하기 위한 대화상자가 열림.


8. 버튼 : 전송하기, 취소하기, 일반 버튼, 이미지 버튼
결제하기, 가입하기, 주문하기....
