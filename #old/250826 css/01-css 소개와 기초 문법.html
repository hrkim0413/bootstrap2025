01-css 소개와 기초 문법

css는 Cascading Style Sheets의 약자로서 html 문서의 내용에 서식을 적용하기 위한 언어로서 미국의 '에릭메이어'라는 개발자에 의해서 개발된 언어이다.

현재까지 css Level3까지 나온 상태로서 웹문서의 'layout', 'effect', 'animation' 등의 동적인 콘텐츠도 개발이 가능한 상태이다.

다양한 사용자 환경에서 동일한 내용을 볼 수 있게 표현해주는 언어로서 웹표준 제작 시 필수이다.

1. css 문법

css 언어는 기본적으로 문서의 상단 head와 head 사이에 style 선언 또는 link 태그를 사용하여 외부서식을 연결 할 수 있다.

css 문법 - 선택자 {속성 : 값;}

  • 선택자(selector) : 서식을 적용하기 위한 하나의 방법, 수단을 말하며 대략 50가지 정도가 있다.
  • 속성(property) : 속성이란? 서식의 종류로서 사용자가 선택한 요소에 '글자색', '배경색', '크기', '여백', '위치', '애니메이션' 등의 내용을 적용하기 위한 서식 내용
  • 값(value) : 속성을 사용하여 어떤 서식을 적용하기 위한 값(내용)을 말함. border: 1px solid #333; font-size : 16px; color: red;
  • : 기호는 속성 : 값 사이에 들어가는 기호 / ; 기호는 속성 : 값; 작성하고 마지막에 들어가는 기호임.

2. css 특징

css는 1)모든 문서에 동일한 서식 적용 가능, 2)어느 환경(pc, tablet, mobile)에서나 사용자가 의도한대로 UI 디자인을 구현할 수 있다. 3)유지보수 측면에서 편리한 장점이 있다.

3. css 주석처리

html과 같이 주석을 사용할 수 있는데 '/* 주석내용 */' 기호로 작성한다.

4. css 작성 시 위치 지정 방법

css 언어 작성 시 상황에 따라 3가지 방법으로 작성 위치를 지정하여 서식을 적용할 수 있다.

  • 내부 스타일 : 현재 html 문서의 상단 head 영역 안에 style 선언하여 작성하는 방법으로서 다른 문서에는 서식을 적용할 수 없다.
  • 외부 스타일 : 별도의 css 문서를 작성하여 html 문서의 head 영역에 link 태그를 사용하여 style 문서를 연결한 다음 서식을 작용하는 방법이며, 여러 html 문서에 동일한 서식을 적용하고자 할 때 유용하며 웹 제작시 가장 많이 사용하는 방법이다.
  • 인라인 스타일 : html 문서의 태그 요소에 직접적으로 스타일 서식을 적용하기 위한 방법으로서 주로 테스트용으로 작성하며, 실제 많이 사용되지는 않는다. 현재 부트스트랩 또는 tailwind css에서 인라인 서식을 많이 사용하고 있음.
