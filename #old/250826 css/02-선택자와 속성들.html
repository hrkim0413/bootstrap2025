02-css 선택자와 속성들

css에서 가장 많이 사용하는 선택자 3가지와 서식을 꾸미기 위한 여러가지 속성들에 대해 알아본다.

1. 기본 선택자 - #id, .class, tag

  • #id 선택자 : id 선택자는 사용자가 선택한 요소에 필요한 서식을 적용하기 위한 선택자이며, 한번 사용된 선택자는 더 이상 사용이 불가하다. 중복되지 않는 서식을 적용하고자 할 때 사용(예 - 레이아웃, 닫기 버튼, 토글버튼 등)
  • .class 선택자 : class 선택자는 사용자가 지정한 요소에 서식을 적용하기 위한 선택자로서 id 선택자와는 다르게 여러번 서식을 적용할 수 있다. '언제든 다시 사용할 수 있는 서식'이라고 할 수 있으며, class 선택자의 이름을 작성 시 '의미있는 이름'으로 작성하며, 이름 앞에 '.' 기호를 넣는 것이 기본이다.
  • tag 선택자 : 지정한 tag를 선택하여 서식을 적용하기 위한 선택자이며, 같은 태그명이 모두 선택이 되기 때문에 실제 사용시 원하는 않는 부분까지도 적용되기 때문에 주의해야 한다.

2. css 속성의 종류

css 서식을 적용하기 위한 속성의 종류는 w3school.com or mdn css를 참고

  • 글자 관련 : font-family(글자 모양), font-size(글자 크기), color(글자 색), text-decoration(밑줄, 줄 없음, 윗줄), text-transform(영문자 대,소문자 전환), font-style(글자를 기울게, 보통) ...
  • 문단 관련 : text-align(문단 정렬), text-indent(들여쓰기), line-height(행간)
  • 크기 관련 : width, height
  • 여백 관련 : margin(바깥 여백), padding(안쪽 여백)
  • 배경 관련 : background-color, background-image, background-size, background-position
  • 테두리 : border
  • 레이아웃 관련 : flex, grid, display, colum
  • 애니메이션 관련 : animation, transition, keyframes
  • 2, 3D 변형 관련 : transform
