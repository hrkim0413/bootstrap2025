03-css속성(property)의 이해

작성법)선택자 {속성 : 값}

  • 선택자(selector) : 서식을 적용하기 위해 대상을 선택하는 방법
  • 속성(property) : 서식의 다양한 종류로서 '속성 : 값;'으로 작성

1. 글자/문단 관련 속성

font-family : 문단의 글자 모양을 지정하고자 할 때 사용하는 속성. 한글 폰트명을 지정하고자 할 때는 '굴림체'처럼 인용부호('')를 붙이며, 영문 글꼴에서는 공백이 있는 폰트명도 인용부호('')를 붙인다.

웹폰트를 사용하면 사용자의 시스템에 글꼴이 없어도 해당 글꼴을 적영하여 보여주게 할 수 있다.

font-size : 글자 크기를 지정하는 속성으로서 px, %, em, rem 단위가 있다. 기본값은 px(고정)단위를 많이 사용. 브라우저의 기본값은 16px임. pc 화면에서는 항상 고정된 크기로 보여야 되기 때문에 'px' 단위를, tablet, mobile 디바이스 화면에서는 해상도가 다르기 때문에 고정이 아닌 유동적인 단위를 사용해야 한다. %, rem 단위를 주고 사용.

color : 글자 색상을 변경하고자 할 때 사용하는 속성으로서 색상값 : 색상명, 색상코드(hex, rgb, rgba, hsl, hsla)를 삽입하여 지정한다.
hex코드 : 0~9, a~f 숫자와 문자를 조합하여 만든 색상코드를 말함. #000000 (검정) #ffffff (흰색) #ff0000 (빨강, R) #00ff00 (초록, G) #0000ff (파랑, B)

font-style : 글자 스타일을 지정할 때 사용하는 속성이며 normal, italic, oblique, inherit 등의 속성값이 있다.

font-weight : 글자를 굵게 하거나 얇게, 보통으로 변경하는 속성, bold, bolder, normal(기본), lighter, 100-900까지 값을 설정이 가능하다.

font-veriant : 영문자를 '작은 대문자'로 변환시켜주는 속성

text-transform : 영문 소문자를 대문자로 전환, 대문자를 소문자로, 단어의 첫글자를 대문자로 전환시켜주는 속성

text-decoration : 글자에 밑줄, 윗줄, 줄없음을 지정하는 속성

2. 문단 관련 속성

text-align : 문단 정렬 속성으로서 left, right, center, justify 속성값이 있음

text-indent : 문단 내용 들여쓰기 속성

line-height : 줄간격(행간) 설정. 기본값 160%
