03 css 선택자_링크 반응 선택자

1. 링크 선택자 - :link, :visited

href 속성을 가진 태그 요소(a), img, 다른 요소에 서식을 적용하기 위한 선택자

선택자 앞에 ':' 기호가 들어가는 이러한 선택자를 '가상 클래스'라고 한다.

  • :link - 링크가 걸린 상태의 요소를 선택하여 서식을 적용
  • :visited - 링크가 걸린 요소를 한번 이상 클릭한 적 있는 경우 서식 적용

2. 반응 선택자 - :hover, :active

  • :hover - 사용자가 마우스로 해당하는 요소에 올렸을 경우 서식을 적용
  • :active - 사용자가 마우스로 해당하는 요소를 클릭하고 있는 경우 서식을 적용
