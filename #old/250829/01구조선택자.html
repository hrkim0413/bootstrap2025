1. .class선택자 : 클래스명앞에 '.'기호를 사용. 재사용하기 위한 서식
  2. #id선택자 : id명 앞에 '#'기호를 사용. 중복되지 않는 서식을 적용하고자 할 때. 레이아웃, 특별한 서식, UX 기능에 대한 서식
  3. tag선택자 : 태그명으로 서식을 적용. 잘못쓰면 기존 서식과 중복되기 때문에 좀심해서 사용
  4. 그룹선택자(,) : A선택자, B선택자를 같이 묶어서 한꺼번에 서식을 적용
  5. 자손, 후손 선택자 : 부모요소와 자식요소와의 관계에서 부모요소 바로 뒤에 오는 요소를 선택, 부모요소 아래 자손, 또한 자손아래 모든 요소 포함하여 선택
  6. 링크, 반응선택자 : a태그요소에 서식을 적용, 사용자가 마우스 올렸을 경우 적용되는 경우 선택

01구조 선택자

구조선택자는 부모와 자식요소 태그 관계에서 자식요소들을 선택하고자 할 때 사용하는데, 자식요소들 중에서 첫째, 마지막 번째, n번째, 짝수번째, 홀수번째를 찾아서 서식을 적용하고자 할 때 많이 사용한다.

구조선택자의 종류
1. 일반구조 선택자
2. 형태구조 선택자(다문화)

1. 일반구조 선택자

  • :first-child : 부모 요소의 첫번째 자손을 선택
  • :nth-child(2) : 부모 요소의 두번째 자손을 선택
  • :nth-child(3) : 부모 요소의 세번째 자손을 선택
  • :last-child : 부모 요소의 마지막 번째 자손을 선택

2. 형태구조 선택자

  • :first-of-type : 형태가 다른 자식요소들 중에서 첫번째를 선택한다.
  • :last-of-type : 형태가 다른 자식요소들 중에서 마지막번째를 선택한다.
  • :nth-of-type(n) : 형태가 다른 자식요소들 중에서 n번째를 선택한다.
  • :nth-of-type(2n) : 형태가 다른 자식요소들 중에서 2n번째를 선택한다.
