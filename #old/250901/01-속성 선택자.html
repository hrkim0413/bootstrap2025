01 속성 선택자

속성 선택자는 html 태그 요소 속성(href, title, src, alt, type...)을 선택하여 서식을 적용하고자 할 때 사용하는 선택자로서 기본 속성, 문자열 속성 선택자가 있다.

1. 기본 속성 선택자

선택자[속성] : 선택자 안에 '속성'이 일치하는 것을 선택한다.

선택자[속성=값] : 선택자 안에 '속성=값'이 일치하는 것을 선택한다.

예)
로그인 폼

2. 문자열 속성 선택자

문자열 속성 선택자는 속성값 안에 문자열에서 사용자가 찾고자 하는 문자를 선택하여 서식을 적용하는 방법

  • 선택자[속성|(파이프라인)=값] : 특정값이 포함된 속성을 선택
  • 선택자[속성^(써컴플렉스)=값] : 시작하는 단어가 포함된 속성을 선택
  • 선택자[속성$(달러사인)=값] : 끝나는 단어가 포함된 속성을 선택
  • 선택자[속성*(아스테리스크)=값] : 문자열 안에 해당 단어가 포함된 속성을 선택
  • 선택자[속성~(틸트)=값] : 여러값들 중 특정 '값'이 포함되어 있을 때 선택
Naver
