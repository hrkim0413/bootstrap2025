01 상태 선택자

  • :checked - 양식을 체크한 상태
  • :focus - 초점이 맞추어진 상태
  • :enabled - 사용이 가능한 요소를 선택
  • :disabled - 사용이 불가능한 요소를 선택

노랑풍선 '여행지 검색 양식', 'id, pw 입력 양식'을 클릭시 테두리 색상 변경됨 => focus(초점)

노랑풍선 로그인 폼 양식 아래 '아이디 저장' 체크박스 클릭시 아래 내용 나오거나 숨기기, cjmall 고객센터 - 질문과 답변 top10, 모바일 토글버튼 클릭시 내비게이션 나오거나/숨기기, 본인인증 페이지에서 약관동의시 버튼 색상 변경 가능하게 할 수 있음 => checked
