클래스 선택자는 다른 클래스 선택자와 중복으로 사용할 수 있는데, 선택자 사용시 '클래스1 클래스2'처럼 적용하며 개수에는 상관이 없다.

2. :root 선택자

:root 선택자는 '의사클래스 선택자'로 문서 트리의 루트 요소를 선택하며, 가상 선택자를 이용해 '재사용할 값을 담아줄 변수값을 저장하는데 사용'한다.

'변수'값을 지정하고자 할 때는 '--'기호를 붙여주어야 한다.

선택자에 적용 방법 : 선택자{속성 : --var(변수명);}

웹페이지에 기준이 되는 스타일 속성값을 사용하고자 할 때 사용하며, 글자, 색상, 레이아웃 관련 서식에 적용한다.
