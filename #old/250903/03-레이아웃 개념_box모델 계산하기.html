03-레이아웃 개념_box모델 계산하기

box model이란? 블럭요소와 인라인요소가 가질 수 있는 '크기'와 '마진', '패딩', '테두리'값을 사용하여 배치하는 개념을 말한다.

박스 계산공식
전체 가로크기 = 가로너비 + 패딩값 + 마진값 + 테두리값
전체 세로크기 = 세로높이 + 패딩값 + 마진값 + 테두리값

패딩값 : padding 10px(위, 오른, 아래, 왼쪽 순으로 적용됨)

padding: 0 10px 20px 30px(12시 방향부터 시계방향으로 돌아가며 값을 적용)

padding: 10px 20px(위/아래 10px, 오른/왼쪽 20px 순으로 적용)

padding-top: 10px;(방향을 지정하여 해당하는 값을 적용)

마진값도 같은 방법으로 설정함.

박스계산 실습

100*100

120*120(패딩값 포함)

120*130

122*141

120*132

140*115(box-sizing: border-box;)
