01-그리드 레이아웃

grid(격자)속성은 flex(플렉스, 가로, 세로, 1차원)속성과는 다르게 두 방향(가로, 세로=2차원)으로 레이아웃을 만들기 위한 시스템이다.

웹페이지 레이아웃 제작방법

  • table - 표, 달력, 게시판, 공지사항, 질문과 답변 테이블...
  • float : left, right
  • display : inline
  • display : inline-block
  • display : block
  • display : table
  • display : flex(1차원, 가로/세로방향 하나만 선택 가능)
  • display : grid(2차원, 가로/세로축 동시에 가능)
  • 다단 레이아웃

1. grid column(열, 세로)

2. grid row(행, 가로)

3. grid gap(여백)

4. grid line(기준선)

5. grid Container(부모속성)

grid layout에서 부모역할을 하기 위한 html태그요소이며 display:grid, display:inline-grid속성 값을 지정할 수 있다.

부모요소의 이름은 'wrap, wrapper, container', 자식요소의 이름은 'item'이라고 하며, 그리드는 'column', 'row'아이템으로 구성된다.

  • display: grid, display: inline-grid : 부모요소에 그리드 속성 적용
  • grid-template-columns : 세로방향으로 나누고자 할 경우 적용하는 속성으로 개수를 지정
  • grid-template-rows : 가로방향으로 나누고자 할 경우 적용하는 속성으로 개수를 지정
  • grid-template : 행과 열을 한꺼번에 나누고자 할 경우 적용하는 속성
  • grid-template-area : 영역의 이름으로 지정하고자 할 경우 적용하는 속성
  • grid-gap, grid-column-gap, grid-row-gap : 행과 열 사이 간격 설정
  • justify-items, align-items, place-items
  • justify-content, align-content, place-content
  • grid-auto-rows, grid-auto-column, grid-auto-flow

6. grid Item(자식에 적용하는 속성들)

grid레이아웃에서 부모요소의 자식요소들은 '아이템'이라고 하며, 아이템들을 병합하고자 할 때, 열, 행번호, 영역의 이름을 따로 지정하여 설정 할 수 있다.

  • grid-column : item의 열 시작위치 번호와 끝번호를 지정하여 병합하고자 할 때 사용. 예)1/5
  • grid-row : item의 행 시작위치 번호와 끝번호를 지정하여 병합하고자 할 때 사용. 예)1/3
  • grid-column, grid-row속성은 각각 grid-column-start, grid-column-end, grid-row-start, grid-row-end속성으로도 사용 가능
  • grid-row-start, grid-row-end
  • grid-column, grid-row
  • grid-area
  • justify-self, align-self, place, self
