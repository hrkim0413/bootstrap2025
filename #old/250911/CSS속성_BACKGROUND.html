CSS속성_BACKGROUND

background속성은 사용자가 선택한 요소에 배경색, 배경 이미지를 삽입하기 위한 속성이다.

1. 배경속성의 종류

  • background-color:배경색(단일컬러, 그라디언트)
  • background-image:배경이미지
  • background-repeat:배경이미지 반복설정
  • background-size:배경이미지 크기설정(px, %)
  • background-position:배경이미지 위치설정(x, y)
  • background-origin:배경이미지 기준위치 지정
  • background-attatchment:배경이미지 고정하기
  • 멀티 배경이미지 삽입 : css3속성에서 추가된 속성으로 여러 이미지를 한꺼번에 배경으로 삽입가능

2. 배경색(background-color) 적용하기

배경색상을 넣기 위한 속성으로서 '색상명', 'hex코드', 'rgb', 'rgba', 'hsl', 'hsla'값의 종류가 있다.

색상값 작성시 alpha의 약자인 a를 사용하면 투명도로 적용이 가능하다. 100% => 1, 50% => 0.5 소수점으로 입력해야 함. 예) background-color:rgba(255,0,0,.5)

이미지 스프라이트

웹페이지를 제작할때 사용하는 아이콘,배너,버튼등을 한개의 그림파일에 넣어놓고 CSS의 background-position속성을 사용하여 해당그림을 보여주는 방식

장점 : 웹(web)에서 불러와야 하는 이미지(image)가 여러개인 경우 그 만큼 서버에서 요청을 해야 하지만, sprite image를 사용하면 한번의 요청으로 image를 보여주게 되므로 서버 대역폭을 줄일 수 있음. => 이미지 로딩지연이 되지 않음.

단점 : 다양한 해상도에 따른 이미지가 있어야 하기 때문에 각 크기별로 또는 별도로 모바일 해상에 맞는 이미지를 따로 제작해야 한다.
