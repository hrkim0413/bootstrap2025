스와이퍼 슬라이드 라이브러리

swiper slide는 자바스크립트 라이브러리 종류의 하나로서 모바일 우선의 강력한 슬라이더/갤러리 플러그인입니다.

웹사이트에서 이미지를 슬라이드하거나 콘텐츠를 좌우로 전환하는 효과를 구현할 떄 많이 사용됩니다. swiper는 다양한 기능을 제공하여 사용자에게 더 나은 경험을 제공할 수 있습니다.

1. siper 설치 방법

  1. cdn 방식 - content delivery network의 약자로 인터넷에서 링크 주소를 복사하여 삽입하여 사용하는 방식으로 간단하게 직접 연결하게 사용할 수 있음. 단점은 네트워크가 연결이 안되면 사용못함.
  2. npm 방식 - 개발자의 시스템 컴퓨터에 직접 소스파일을 설치하여 사용하는 방식

2. 주요옵션

  • loop: 슬라이드가 끝나면 처음으로 돌아가는 무한 루프 설정
  • autoplay: 자동으로 슬라이드가 넘어가도록 설정, delay로 시간 간격을 설정
  • navigation: 좌/우 버튼을 설정
  • pagination: 페이지네이션==콘트롤버튼(슬라이드 번호)을 설정
  • effect: 슬라이드 효과 종류 (slide, fade, cube, coverflow 등)
  • speed: 슬라이드 전환 속도 설정

3. 스와이퍼 슬라이드 스타일 적용하기

swiper 플러그인은 기본적인 css 스타일이 있지만 사용자가 직접 별도로 css서식을 만들어서 적용할 수 있습니다

필요에 따라 다양한 효과나 옵션을 조정하여 사용자가 더 매력적이고 직관적인 사용자 경험을 느낄 수 있습니다

4. swiper 슬라이드 삽입하기

  1. swiper css파일 연결
  2. html 태그 작성
  3. 페이지 하단에 script 작성
  4. 기타 - 사용자가 원하는 스타일 또는 기능 편집
