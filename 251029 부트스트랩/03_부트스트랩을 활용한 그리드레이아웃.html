03_부트스트랩을 활용한 그리드레이아웃

부트스트랩은 하나의 행에 반드시 12개의 열(컬럼)을 사용해야 합니다.

만약 12개의 열(컬럼)을 모두 사용하지 않으려면, 더 넓은 열(컬럼)을 만들기 위해 열(컬럼)들을 하나의 그룹으로 만들어 사용할 수 있습니다.

그리드 시스템에서는 화면의 너비별 5개의 클래스를 가지고 있습니다. 이미지 참고

각 해상도별 클래스명 이해하기

  • 해상도 576미만일 경우 클래스명 없음
  • 해상도 576이상일 경우 클래스명 'sm' - 모바일 버전
  • 해상도 768이상일 경우 클래스명 'md' - 태블릿 작은 버전
  • 해상도 992이상일 경우 클래스명 'lg' - 태블릿 큰 버전
  • 해상도 1200이상일 경우 클래스명 'xl' - 피씨 일반 버전
  • 해상도 1400이상일 경우 클래스명 'xxl' - 피씨 큰 버전

그리드 레이아웃 실습

12컬럼을 한줄에 배치

lg: 가로해상도 992px이상일 경우 가로영역을 12컬럼으로 인식(설정)

12컬럼을 4칸씩 병합하여 배치
12컬럼을 4칸씩 병합하여 배치
12컬럼을 4칸씩 병합하여 배치

md: 가로해상도가 768px이상일 경우 가로영역을 3등분(4칸을 1칸으로 병합)으로 인식

12컬럼을 6칸씩 병합하여 배치
12컬럼을 6칸씩 병합하여 배치

sm: 가로해상도가 576px이상일 경우 가로영역을 2등분(6칸을 1칸으로 병합)으로 인식

실습예제. 스타벅스 음료 페이지의 상품목록 그리드 레이아웃 설정하기

가로해상도 992px이상일 경우 4칸씩 배치
가로해상도 768px이상일 경우 3칸씩 배치
가로해상도 768px미만일 경우 2칸씩 배치
